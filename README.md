These data files support figures in our manuscript : Hobaiter C, Dominy NJ Firelight and the origins of spoken language

Quantifying firelight with a SpectraScan PR-670 spectroradiometer (Photo Research, Syracuse, USA) at a distance of 1 m and height of 75 cm. We provide the raw Firelight data (Figure 1 panels) for flames and coals, as well as the calculated data for luminance.

We also provide the Flickerate data (Figure 2 panels) as separate RGB channels. We quantified the flicker rate of firelight using the entire flame at 500 fps using a high-sensitivity colour camera (FASTCAM SA1.1; Phototron, Tokyo, Japan). Postprocessing of slow-motion video was performed with custom software built in MatLab. In panel (A) Each frame was separated into RGB colour-space and the image intensity of each pixel quantified, weighted, and averaged. In panel (B) Frame intensity was plotted as a function of time in frequency space using a Fourier transform. The code available to support plotting these data is also provided.
